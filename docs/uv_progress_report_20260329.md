# UV 映射阶段汇报（截至 2026-03-29）

## 0. 一句话结论
我们已经把“低模复用高模贴图”从一个能跑的线性流程，推进到了“有拓扑切缝、可分岛求解、可做非线性可逆精修、并且有完整诊断数据”的工程体系。  
目前最核心的现实是：几何稳定性明显提升了，但颜色一致性还没有同步提升。

---

## 1. 问题定义与评价标准

### 1.1 我们在解决什么问题
输入有两部分：
1. 一个高模，带原始 UV 和原始贴图。
2. 一个低模，几何简化后没有可用 UV。

目标是输出低模 UV，让低模直接使用高模贴图时，尽量接近高模观感。

### 1.2 为什么这个问题难
低模比高模粗糙很多，一个低模大三角形往往覆盖高模上大量细节。  
如果对应关系不稳，或者跨过了高模 UV 岛边界，求解器就会被迫“用一个平滑插值去拟合不连续目标”，出现拉丝、液化、纹理错位。

### 1.3 我们主要看哪些指标
1. 坏三角比例：UV 畸变严重的三角形占比。
2. 翻转比例：UV 面积为负或接近退化的三角形占比。
3. 颜色一致性（L1/L2）：把低模预测 UV 和目标 UV 在贴图上采样后，比较颜色误差。

指标含义是：
1. 前两项偏几何合法性。
2. 第三项偏视觉贴图对齐。
3. 三项要一起看，不能只看一项。

---

## 2. 第一阶段：线性全局拟合（普通 PCG 路线）

### 2.1 采样是怎么做的
不是只看顶点，而是在每个低模三角形内部采样多个点。  
每个面的采样数按面积自适应：
\[
n_f=\text{clip}\Bigl(\text{round}\bigl(\text{base}\cdot\sqrt{A_f/\bar A}\bigr),n_{\min},n_{\max}\Bigr)
\]
这意味着大面采样更多，小面采样更少。  
每个采样点用随机重心坐标生成，保证在三角形内均匀分布。

### 2.2 高模对应位置怎么找
对每个低模采样点，沿法线正反两个方向发射射线找高模交点。  
如果射线失败，再用最近点回退。  
保留法线方向合理、距离合理的命中，得到“这个采样点在高模上的目标 UV”。

### 2.3 线性系统怎么建
未知量是低模所有顶点的 UV。  
每个采样点给出一个“重心插值应接近目标 UV”的约束：
\[
\hat u_s = w_0 u_{i}+w_1 u_{j}+w_2 u_{k}\approx u_s^\star
\]
把所有采样点叠起来，就是加权最小二乘：
\[
\min_{\mathbf{x}}\sum_s \omega_s\|\mathbf{B}_s\mathbf{x}-\mathbf{t}_s\|^2
\]
再加平滑、锚点和微小对角正则，形成稳定系统：
\[
(\mathbf{A}^\top\mathbf{A}+\lambda\mathbf{L}+\mathbf{H}_{anchor}+\epsilon\mathbf{I})\mathbf{x}
=\mathbf{A}^\top\mathbf{b}
\]
最后用 PCG 或稳健线性后端求解。

### 2.4 这个方法为什么会“拉丝”
它本质上是“全局线性平滑拟合”。  
当一个低模面内部对应到多个不连续区域（比如跨 UV 岛）时，线性模型会把不连续目标硬拉成连续，于是面内出现很强拉伸。

## 3. 你做过的关键增强：UV 岛与切缝体系

这部分是目前工程里非常关键的进展。

### 3.1 高模分岛
先在高模面级别识别 UV 岛。  
判定逻辑是：共享几何边的两侧面，只有当边两端 UV 也连续时，才属于同一岛。

### 3.2 低模面打“主岛标签”
把低模每个面的采样命中统计成投票，得到“这个低模面最可能属于哪个高模岛”。  
同时标记：
1. 冲突面：一个低模面明显命中多个岛。
2. 未知面：有效样本不足或不稳定。

### 3.3 半边切缝
对跨岛边进行切缝：复制顶点，断开拓扑连通。  
这不是显示层“假切”，而是求解网格拓扑真的分开了，接缝两侧可以独立求解 UV。

### 3.4 严格过滤
在进入求解前，额外做两层过滤：
1. 面内 UV 跨度过大的样本剔除。
2. 与面主岛不一致的样本剔除。

这一步直接抑制了“跨岛污染”。

---

## 4. 第二阶段：方案2（梯度传递 + 泊松重建）

### 4.1 为什么从点拟合转向梯度拟合
点对点拟合容易受局部对应噪声影响。  
方案2改成先拟合“局部变化趋势”，再整体积分重建，目标是让 UV 形变更平顺、更几何一致。

### 4.2 梯度目标如何得到
1. 先在高模每个面计算局部“UV 对 3D 的变化率”（可理解为每个面的局部变换）。
2. 对每个低模面，把命中的高模面变化率做鲁棒聚合（有离群剔除和迭代重加权）。
3. 我们后续又加入了“尺度补偿”，避免聚合后幅值被过度抵消。

### 4.3 泊松重建怎么做
聚合后，每个低模面都有一个目标局部变化率。  
对面内三条边，把“目标边 UV 增量”写成：
\[
\Delta \mathbf{u}_{edge}\approx \mathbf{J}_{face}\cdot \mathbf{e}_{3D}
\]
然后把所有边约束汇总成线性系统，继续配合平滑与锚点求解。  
直观上，这相当于先规定“每条边应该怎么变”，再找一个全局 UV 场尽量满足这些边差分关系。

### 4.4 与第一阶段相比，方案2的增量
1. 约束对象从“采样点目标位置”变成“面/边的局部变化率”。
2. 增加了鲁棒聚合和离群过滤。
3. 默认支持按岛拆子问题求解。
4. 增加了大量诊断数据，便于定位失真来源。

### 4.5 方案2当前表现
在 massive 数据上（64 和 128 两档）：
1. 坏三角和翻转明显下降。
2. 颜色 L1 反而上升（大约退化 13%~17%）。

这说明它更偏向“几何稳定优先”，但“贴图细节对齐”还没跟上。

## 5. 第三阶段：方案4（非线性可逆精修）

### 5.1 方案4到底在做什么
方案4不是重做整条链，而是“拿方案2结果做非线性精修”。  
核心目标是：把翻转/退化压下去，让映射更接近可逆。

### 5.2 可逆性如何体现
对每个三角形，关注 UV 映射的局部面积变化（局部雅可比行列式）。  
若面积变成负值或接近 0，就说明翻转或塌陷。  
方案4通过能量项持续惩罚这种情况，从机制上把解推回正面积区域。

### 5.3 非线性能量由哪些部分组成
总能量可以理解为“多目标加权和”：
1. 数据项：保持与方案2目标局部变化率一致。
2. 可逆项：对接近奇异或翻转的三角形给强惩罚。
3. 形变项：惩罚过度拉伸和过度压缩（对称保形思想）。
4. 平滑项：抑制局部高频噪声。
5. 锚点项：限制整体漂移。

### 5.4 怎么优化
1. 以方案2结果为初值。
2. 先做一轮翻转预修复，避免一开始就在不可行区域。
3. 用 L-BFGS（可切 Adam）迭代下降。
4. 用回溯线搜索、梯度裁剪、屏障同伦升温来稳住迭代。
5. 如果仍然违规或不收敛，触发回退到方案2。

### 5.5 与方案2的增量差异
1. 方案2是线性闭式风格，快且稳，但可逆约束弱。
2. 方案4是非线性约束优化，目标更强，但稳定性更难。
3. 方案4理论上更“像论文方法”，但工程上更依赖初始化和参数。

### 5.6 方案4当前表现
目前在这批实验中，经常出现“非线性阶段未稳定超越，最后回退到方案2”。  
所以现阶段它更像“上限探索模块”，还不是稳定主力。

## 10. 数学细节补充（给技术评审）

### 10.1 线性点拟合模型（第一阶段）
对每个采样点，预测 UV 是所在低模面三个顶点 UV 的重心插值。  
把所有采样点写成矩阵形式后，目标函数可写成：
\[
\min_{\mathbf{x}}
\|\mathbf{W}^{1/2}(\mathbf{A}\mathbf{x}-\mathbf{b})\|_2^2
+\lambda\|\mathbf{L}\mathbf{x}\|_2^2
+\alpha\|\mathbf{C}(\mathbf{x}-\mathbf{x}_{anchor})\|_2^2
+\epsilon\|\mathbf{x}\|_2^2
\]
其中：
1. 第一项是数据拟合。  
2. 第二项是网格平滑。  
3. 第三项是锚点稳定。  
4. 第四项是数值稳定。  

法方程为：
\[
(\mathbf{A}^\top\mathbf{W}\mathbf{A}+\lambda\mathbf{L}^\top\mathbf{L}+\alpha\mathbf{C}^\top\mathbf{C}+\epsilon\mathbf{I})\mathbf{x}
=
\mathbf{A}^\top\mathbf{W}\mathbf{b}+\alpha\mathbf{C}^\top\mathbf{C}\mathbf{x}_{anchor}
\]

### 10.2 方案2的“梯度传递 + 泊松重建”模型
方案2把“点位置拟合”改成“边差分拟合”。

第一步是构建每个低模面的目标局部变换。  
一个低模面对应多个高模命中样本，每个样本给一个局部变换，先做鲁棒聚合。  
当前使用的是带离群剔除的迭代重加权，核心思想是：
1. 先算加权均值。  
2. 根据残差给离群样本更小权重。  
3. 重复几轮得到更稳的面级目标。  

第二步是把面级目标变成边约束。  
对每个面三条边：
\[
\mathbf{u}_j-\mathbf{u}_i \approx \Delta \mathbf{u}_{ij}
\]
\[
\Delta \mathbf{u}_{ij} = \mathbf{J}_{face}\mathbf{e}_{ij}
\]
这就变成了标准 Poisson 风格的“梯度积分”问题：
\[
\min_{\mathbf{x}} \sum_{edges} w_{ij}\|(\mathbf{u}_j-\mathbf{u}_i)-\Delta \mathbf{u}_{ij}\|^2
+ \text{regularization}
\]

第三步是全局求解并做漂移修正。  
求解后再估计整体平移偏差，做一次平移对齐，减少全局错位。

### 10.3 方案4的非线性精修模型
方案4以方案2输出为初值，优化目标是“既贴近目标，又不翻转”。  
总能量是多项加权和：
\[
E=E_{data}+E_{smooth}+E_{sd}+E_{barrier}+E_{flip}+E_{anchor}
\]
各项含义：
1. \(E_{data}\)：保持与方案2面级目标一致。  
2. \(E_{smooth}\)：抑制局部噪声。  
3. \(E_{sd}\)：对称保形项，同时惩罚过拉伸和过压缩。  
4. \(E_{barrier}\)：惩罚局部面积接近 0。  
5. \(E_{flip}\)：对负面积直接加罚。  
6. \(E_{anchor}\)：限制整体漂移。  

优化方法使用拟牛顿或一阶法，并配套：
1. 初始翻转预修复。  
2. 回溯线搜索。  
3. 梯度裁剪。  
4. 屏障权重同伦升温。  
5. 局部补丁修复。  

“可逆性”就体现在：如果某个三角形要翻转或塌陷，能量会急剧变大，优化会主动把它推回正面积区域。

## 11. 三分钟口头汇报稿（可直接读）

### 11.1 开场（30 秒）
我们的目标是让低模直接复用高模贴图。  
难点是低模简化后，一个大三角形会覆盖高模很多细节，导致传统线性拟合很容易出现拉丝和错位。

### 11.2 我做了什么（60 秒）
第一步，我先把基础线性流程打通，包括采样、对应、全局求解和导出。  
第二步，我把 UV 岛处理做成了完整体系：高模分岛、低模投票打标签、半边切缝复制顶点、严格跨岛过滤。  
第三步，我实现了方案2，用梯度传递加泊松重建替代点拟合，并做了分岛求解和完整诊断。  
第四步，我实现了方案4，在方案2结果上做非线性可逆精修，加入了翻转预修复、线搜索和回退保护。

### 11.3 结果（45 秒）
在 massive 数据上，方案2和方案4都显著降低了坏三角和翻转比例。  
但颜色一致性目前仍落后旧基线，大约退化 13% 到 17%。  
方案4目前经常回退到方案2，说明非线性约束还没稳定释放收益。

### 11.4 结尾（45 秒）
当前阶段的核心价值是：我们已经从“能跑”进入“可解释、可诊断、可迭代优化”的状态。  
下一步重点不是盲目调参，而是把评价体系做正确，并针对方案2的细节保真和方案4的收敛稳定性做有证据的改进。
